Wordle but worse in many ways

The proposal is to implement a working version of the wordle game framework as a web application.

Wordle is a game written by an independent developer and recently purchased by NY Times. The basic premise is a simple information game where the player makes a limited number of guesses about what a word will be. An indication on the correctness of their guess is provided to improve further guesses.
